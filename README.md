# i18n
js 多语言处理

数据结构 json:

*en..js*
```
    {
        "str1":"11",
        "test":"TEST",
    }
```

*php路径格式*

```
    "en":{
        "str1":"11",
        "test":"TEST",
    }
    "zh":{
        "str1":"一一",
        "test":"测试",
    }
```

# 功能

支持文件、路径加载。

路径加载返回值包括全语种全部数据。

文件加载仅包括对应语种全部数据。

可适用于多语言显示，也可适用于当用户数据时显示字段对应变量显示。

# 运行

php ： php -S localhost:80
